What's New in CopyTeX v1.5.1

In-page export, prompt management, and a polished new look — all designed to supercharge your AI workflow.

New Features
New

In-Page Export Button

A floating Export Chat button now appears at the bottom-right corner of every supported AI chat page. Click it to open a beautiful export modal — no need to open the popup.

New

Export All Chats as ZIP

Select All Chats in the export modal to bundle the current conversation and a full sidebar conversation index into a single .zip file. Choose Markdown, JSON, or both formats.

New

Custom Prompt Modal

Adding prompts now uses a beautiful in-page dialog with name and content fields, validation, and keyboard shortcuts (Ctrl+Enter to save, Esc to cancel) — replacing the old browser prompt.

New

Export Scope Selection

The export modal lets you choose between Current Chat (full message content) and All Chats (sidebar conversation index + current content), with real-time message counts.

New

Gemini Watermark Remover

Automatically removes the semi-transparent watermark from Gemini-generated images using a mathematically precise Reverse Alpha Blending algorithm — lossless, instant, and fully client-side.
